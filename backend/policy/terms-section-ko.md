## 제 N 조 (크로마이핑 사용자 등록 콘텐츠)

회원이 「크로마이핑 팩 마켓플레이스」에 사운드 팩을 등록하는 행위는 본 조항에 동의하는 것으로 간주됩니다.

### ① 인덱서 모델

회사(우수에프앤씨 주식회사, 이하 "회사")는 「크로마이핑 팩 마켓플레이스」의 인덱서로서, 회원이 등록한 사운드 콘텐츠를 회사의 서버에 직접 호스팅하지 않습니다. 콘텐츠의 다운로드는 회원이 지정한 GitHub 등 외부 저장소에서 직접 이루어집니다.

### ② 등록자의 책임

1. 회원은 등록한 모든 콘텐츠의 저작권을 본인이 보유하거나, 정당한 라이선스를 가지고 있음을 보증합니다.
2. 등록한 콘텐츠의 저작권 침해, 명예훼손, 개인정보 침해 등 법적 문제가 발생할 경우, 모든 법적 책임은 등록한 회원에게 있습니다.
3. 회사는 회원이 등록한 콘텐츠의 사전 검토 의무를 지지 않습니다.

### ③ 등록 금지 콘텐츠

다음 콘텐츠는 등록할 수 없습니다:

1. 본인이 권리를 보유하지 않은 게임/영화/드라마/방송 등의 음원
2. 타인의 음성을 본인 동의 없이 사용한 콘텐츠
3. 성인용·폭력적·혐오적·차별적 콘텐츠
4. 광고 또는 스팸성 콘텐츠
5. 악성 코드, 멀웨어, 또는 시스템에 해를 끼치는 콘텐츠
6. 그 밖에 대한민국 법령을 위반하는 콘텐츠

### ④ 신고 및 처리 절차

1. 권리침해 또는 부적절한 콘텐츠 발견 시, 누구나 신고할 수 있습니다.
2. 회사는 신고 접수 후 24-48시간 이내에 검토 및 처리합니다.
3. 정당한 신고로 확인된 경우, 회사는 즉시 해당 콘텐츠를 비공개 또는 영구 제거합니다.
4. 등록자는 처리 결과에 대해 통지받으며, 이의신청을 할 수 있습니다.

### ⑤ 반복 위반자에 대한 조치

1. 1회 위반: 경고 통지
2. 2회 위반: 30일 등록 정지 및 해당 콘텐츠 제거
3. 3회 위반: 영구 계정 차단 및 모든 등록 콘텐츠 제거

### ⑥ 본인 확인 정보의 활용

회원가입 시 제공한 본인 확인 정보(휴대전화 번호 등)는 콘텐츠 관련 분쟁 발생 시 법적 절차(수사 협조, 민사 소송 등)에 활용될 수 있습니다.

### ⑦ 라이선스 부여

회원은 등록한 콘텐츠에 대해 회사 및 다른 사용자에게 다음 권리를 부여합니다:

1. 메타데이터(이름, 설명, 카테고리 등)를 회사가 인덱싱·표시·검색 결과에 포함시킬 권리
2. 다른 사용자가 크로마이핑 클라이언트를 통해 등록자의 GitHub 저장소에서 콘텐츠를 다운로드할 수 있도록 안내할 권리

회원은 언제든지 콘텐츠를 삭제하여 위 권리 부여를 철회할 수 있습니다.

### ⑧ 회사의 권리

회사는 다음의 권리를 가집니다:

1. 본 조항을 위반하거나 위반이 의심되는 콘텐츠를 사전 통지 없이 비공개 처리할 권리
2. 본 조항 ③항의 등록 금지 콘텐츠로 판단되는 경우 영구 제거할 권리
3. 회사가 운영하는 「크로마이핑 팩 마켓플레이스」의 운영 정책을 변경할 권리

### ⑨ 면책 조항

1. 회사는 회원이 등록한 콘텐츠로 인해 다른 사용자에게 발생한 손해에 대해 책임지지 않습니다.
2. 회사는 외부 저장소(GitHub 등)의 가용성, 다운로드 속도, 콘텐츠 보전 등에 대해 보증하지 않습니다.
3. 회사는 자동화된 검증(파일 형식, 크기 등)을 수행하나, 콘텐츠의 저작권 합법성을 보증하지 않습니다.

## 신고 처리 SOP (사내 운영 가이드)

이 부분은 사용자에게 보이지 않는 **운영 가이드**입니다.

### 처리 우선순위
1. **저작권 침해 신고 (DMCA / 권리자 직접 신고)**: 24시간 내
2. **음란/혐오 콘텐츠**: 24시간 내
3. **악성 코드 의심**: 즉시 (자동 비공개)
4. **저품질/스팸**: 48시간 내
5. **기타**: 72시간 내

### 처리 단계
1. 신고 접수 → 자동 알림 (이메일/Slack)
2. 관리자 검토 → 정당성 판단
3. 정당한 신고 → 콘텐츠 비공개 (`status='frozen'`)
4. 등록자에게 통지 (이메일)
5. 7일 이의신청 기간
6. 영구 제거 (`status='removed'`) 또는 복구 (`status='active'`)
7. 신고자/등록자에게 결과 통지

### DMCA 대응 표준 양식

권리자가 신고할 때 다음 정보 요구:
- 권리자 이름 + 연락처
- 침해 콘텐츠 URL (cromaizing.com/cromaiping/packs/[name])
- 권리 증명 자료 (저작권 등록증, 라이선스 등)
- 본인 또는 권한 있는 대리인 확인서
- 위증 시 처벌 동의 (서면)

### 반복 위반자 추적

`pack_metadata.submitter_user_id`로 같은 사용자 추적:
- 회원 프로필에 위반 이력 표시 (관리자만 보이게)
- 3회 누적 시 자동 영구 차단 (트리거 또는 cron job)
